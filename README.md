# UMat Test
A test program of ordinary Mat and UMat of OpenCV 3.x.

# How to Compile and Run
This project includes two program: ```normal``` and ```umatTest``` <br>
For ```normal```
```
$ cd normal
$ mkdir build
$ cd build
$ cmake ..
$ make
$ cd ..
$ ./normal
```

For ```umatTest```
```
$ cd umatTest
$ mkdir build
$ cd build
$ cmake ..
$ make
$ cd ..
$ ./umatTest
```

# Note
I use OpenCV with version 3.4.1.
